# UI Load & Performance Framework (Python + Pytest + Playwright + Locust)

This repo provides a UI-only load & performance testing framework:
- Playwright + Pytest for deterministic E2E journeys.
- Locust for load shaping and distributed execution.
- Playwright MCP to generate initial journey drafts (then harden).
